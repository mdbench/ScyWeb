Welcome to ScySDK Docs!

Select an assessment or language-specific SDK documentation from the sidebar to begin technical review.

The ScySDK Documentation Hub provides a bifurcated interface designed to bridge the gap between theoretical security assessments and practical implementation. On the primary navigation sidebar, users will find a clear distinction between the Assessment Section and the SDK Documentation. The Assessment area is a repository of mathematical deep-dives, comparing the ScySDK geometric manifold against legacy relational and document-based databases. These technical whitepapers serve to validate the architectural superiority of the Hilbert-curve logic and provide the empirical evidence required for infrastructure verification and patent protection strategies.

The secondary section of the hub is the SDK Documentation Tree, a language-agnostic integration guide organized by local directory structures. By navigating the collapsible language nodes — which support ten distinct environments from C++ and Rust to React Native and Swift — developers can access specific implementation modules.

To utilize this portal effectively, simply select an analysis from the Assessment section to view the structural and security performance metrics of the image-based database. For active development, drill down into the language folder corresponding to your project stack to find detailed guides on utilizing the ScySDK "vines" for high-performance data retrieval.
